# SkillOpt for system-cli-subagents

Це робочий каркас постійного поліпшення skill-доків цього проєкту.

## Мета

- покращувати `freebuff-worker-bridge/SKILL.md`
- покращувати `launch-system-cli-agent/SKILL.md`
- тримати `best_skill.md` як поточний промотований варіант

## Артефакти

- `skillopt/best_skill.md` - поточна найкраща версія інструкцій
- `.agents/plugins/system-cli-subagents/skills/skillopt-self-improvement/SKILL.md` - skill для запуску циклу покращення

## Правило прийому змін

Приймай лише ті правки, які:

- проходять dry-run `freebuff2api`
- не обіцяють native subagent runtime, якщо його немає
- не ховають fallback script
- роблять інструкції коротшими, яснішими або точнішими

## Коли оновлювати

- після реального збою
- після уточнення від користувача
- після зміни контракту `freebuff2api`
- після появи кращого формулювання, яке проходить перевірку

## Що не робити

- не додавати вигадані можливості
- не зберігати довгі дублікати інструкцій
- не просувати зміни без валідації
